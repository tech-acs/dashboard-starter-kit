---
type: report
title: EAs Performing Below Minimum Threshold
description: Enumeration areas with completion rates below the 10% minimum threshold
---

Lists enumeration areas (EAs) whose household enumeration completion rate falls below
the minimum acceptable threshold of 10%.

Calculated by dividing the number of households enumerated by the EA's assigned target,
then filtering for EAs where the result is below 0.10 (10%). Report should display EA
code, target count, actual count, completion percentage, and number of days since start.

Renders as a table sorted by completion percentage (ascending) for prioritisation,
with an alert-style visual indicator.
