type: report
title: Cases Outside Listing Duration
description: Records listed outside the expected listing exercise duration
---

Lists structure or household records where the listing timestamp falls outside the expected listing start and end dates.

Queries for records where the listing date is before the official start date or after the official end date. Report should display structure/household ID, EA code, listing timestamp, enumerator ID, and the expected date range.

Renders as a table for data quality review, flagging potential data integrity issues.
